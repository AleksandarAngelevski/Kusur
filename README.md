💸 Kusur — Bill Splitting Web App
A web application for splitting expenses among friends and groups, inspired by Splitwise. Built with Spring Boot, Thymeleaf, and PostgreSQL.

✨ Features

Group management — create groups and invite friends
Expense tracking — log shared bills and who paid
Automatic splitting — divide expenses equally or by custom amounts
Balance overview — see who owes what at a glance
Settle up — mark debts as paid and keep balances clean
